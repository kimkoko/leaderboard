About Project

This leaderboard is the final output from the capstone project 'Knowledge Graph Based Question Answering Leaderboard' by Kim Songi (19085436D), describing the overall rank of the knowledge graph based simple question answering models.

The final rank is calculated with the formula from Fair Benchmark for Unsupervised Node Representation (2022). The calculation process is described in the calculation section below.

Following process is the calculation step to produce the overall rank. The lower value means the higher rank.

(Numerical values of log: log 1 = 0, log 2 = 0.301, log 3 = 0.447)

  • KEQA = log 1 + log 1 + log 3 = 0.447           Rank 1
  • BertQA = log 3 + log 2 + log 1 = 0.748         Rank 2
  • BuboQA = log 2 + log 3 + log 2 = 1.049         Rank 3
